# 代码解析

## 相关文件解析

### bjarne配置

**存放位置**：~/.bjarne。

| 文件                 | 说明                                               |
| -------------------- | -------------------------------------------------- |
| last_update_check    | 最新更新检查，里面存放的是上次更新时间戳，单位秒。 |
| auto_update_disabled | 如果这个文件存在，则不自动更新。                   |
